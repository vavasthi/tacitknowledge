Commons-Collections with Generics

This package is a new version of the popular Jakarta Commons-Collections project that introduces support for Java 1.5 Generics. Generics introduce a whole new level of usability and type-safety to the Commons-Collections classes. The functionality of the package is otherwise left unchanged.

See the home page of the project for more information.

The generics conversion was performed by Matt Hall and John Watkinson. See the here for the list of authors of the original Jakarta Commons-Collections.
